# Getting started

# DruxtJS

DruxtJS is a suite of modules to connect a Drupal (opens new window) JSON:API backend to a Nuxt.js (opens new window) frontend.

This module is included in the core DruxtJS (opens new window) project, the below instructions are only required if installing this module standalone.

# Quickstart

  1. Install the NPM module:

    npm i druxt-views
    1

  2. Add the module and configure as required in nuxt.config.js:

    module.exports = {
  modules: [
    'druxt-views'
  ],
  druxt: {
    baseUrl: 'https://demo-api.druxtjs.org'
  }
}
