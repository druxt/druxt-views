# DruxtJS Views

The DruxtJS Views module adds Drupal (opens new window) Views support to your Nuxt.js (opens new window) frontend.

# How it works?

Upon installation the module installs a Nuxt.js plugin that registers the provided Vue.js components.

# DruxtView component.

The DruxtView component fetches the View and View results JSON:API resources and renders the output using the DruxtJS Entity (opens new window) component.

This component can be used directly by the DruxtJS Router (opens new window) to render View pages.

<DruxtView :displayId="displayId" :uuid="uuid" :viewId="viewId" />
1

See the DruxtView component API documentation.

# DruxtBlockViewsBlock component.

The DruxtBlockViewsBlock component adds Views blocks support to the DruxtJS Blocks (opens new window) component.

This component is intended to be rendered by the component.

See the DruxtBlockViewsBlock component API documentation.

Example

<DruxtBlock :uuid="uuid" />
