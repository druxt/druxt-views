# Druxt.js Views

The Druxt.js Views module adds Drupal Views support to your Nuxt.js frontend.

# How it works?

Upon installation the module installs a Nuxt.js plugin that registers the provided Vue.js components.

# DruxtView component.

The DruxtView component fetches the View and View results JSON:API resources and renders the output using the Druxt.js Entity component.

This component can be used directly by the Druxt.js Router to render View pages.

<druxt-view :displayId="displayId" :uuid="uuid" :viewId="viewId" />
1

See the DruxtView component API documentation.

# DruxtBlockViewsBlock component.

The DruxtBlockViewsBlock component adds Views blocks support to the Druxt.js Blocks component.

This component is intended to be rendered by the component.

See the DruxtBlockViewsBlock component API documentation.

Example

<druxt-block :uuid="uuid" />
