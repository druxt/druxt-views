# DruxtViews

The DruxtViews module adds Drupal Views support to your Nuxt.js (opens new window) frontend.

# Features

  • Nuxt module
  • DruxtView component
  • DruxtRouter support
  • DruxtBlock support
  • DruxtWrapper theming system
  • Auto-generated Nuxt Storybook integration

# Nuxt module

The DruxtView module provides Vue.js components, a Vuex store and a Nuxt module, among other things.

The Nuxt module installs all required components and dependencies, however most components can be used individually in any Node project.

module.exports = {
  modules: ['druxt-views'],
  druxt: { baseUrl: 'https://demo-api.druxtjs.org' },
}

# DruxtView component

The DruxtView component fetches the View and View result JSON:API resources and renders the output using the DruxtEntity (opens new window) component.

<DruxtView :displayId="displayId" :viewId="viewId" />
1

See the DruxtView component API documentation.

# DruxtWrapper theming

Druxt modules use a slot-based Wrapper component system to provide rich defaults while still allowing full control over all theming and functionality.

See the theming guide (opens new window) for more details.

# Router support

Adds Views Page support via the DruxtRouter module (opens new window) and the Decoupled Router module (opens new window).

# Blocks support

Adds Views Blocks support via the DruxtBlocks module (opens new window) and the <DruxtBlockRegion /> component

# Storybook

DruxtViews provides zero-config, auto generated Storybook integration with a live data connnection to your Druxt backend.

See Gettings started for more information.
