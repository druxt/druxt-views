# DruxtViewsSortsMixin

Provides Vue.js properties for DruxtViewSorts slot themable components.

Kind: global mixin
Example

<script>
import { DruxtViewSortsMixin } from 'druxt-views'

export default {
  mixins: [DruxtViewSortsMixin],
}
</script>
