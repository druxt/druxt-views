# DruxtViewsViewMixin

Provides Vue.js properties for DruxtView slot themable components.

Kind: global mixin
Example

<script>
import { DruxtViewsViewMixin } from 'druxt-views'

export default {
  mixins: [DruxtViewsViewMixin],
}
</script>
