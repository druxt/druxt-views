# DruxtViewsFiltersMixin

Provides Vue.js properties for DruxtViewsFilters slot themable components.

Kind: global mixin
Example

<script>
import { DruxtViewsFiltersMixin } from 'druxt-views'

export default {
  mixins: [DruxtViewsFiltersMixin],
}
</script>
