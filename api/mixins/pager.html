# DruxtViewsPagerMixin

Provides Vue.js properties for DruxtViewPager slot themable components.

Kind: global mixin
Example

<script>
import { DruxtViewsPagerMixin } from 'druxt-views'

export default {
  mixins: [DruxtViewsPagerMixin],
}
</script>
