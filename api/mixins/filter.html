# DruxtViewsFilterMixin

Provides Vue.js properties for DruxtViewsFilter slot themable components.

Kind: global mixin
Example

<script>
import { DruxtViewsFilterMixin } from 'druxt-views'

export default {
  mixins: [DruxtViewsFilterMixin],
}
</script>
