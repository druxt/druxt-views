# PropsData : object

Provides propsData for the DruxtWrapper.

Kind: global typedef

Param Type Description
options object The Exposed form options.
sorts Array.<object> The Exposed Sort objects.
type string The Exposed form type.
value integer The DruxtViewSorts model value.

Example

{
  options: {
    expose_sort_order: true,
    expose_sorts_label: 'Sort by',
    reset_button: false,
    ...
  },
  sorts: [{
    admin_label: '',
    expose: {},
    exposed: true,
    ...
  }],
  type: 'basic',
  value: undefined,
}
