DruxtJS Views

Drupal Views Vue.js components for Nuxt.js

Get started

The DruxtViews module adds Drupal (opens new window) Views support to your Nuxt.js (opens new window) application.

# The DruxtView component

Druxt Views provides a Vue.js component to render a Drupal View.

<DruxtView :display-id="displayId" :uuid="uuid" :view-id="viewId" />
1

Get started with the Guide and API Documentation.

# Druxt module/wrapper

The DruxtView component is compatible with the DruxtJS component theming system.

Example: DruxtEntityViewViewId.vue

<template>
  <div v-if="slot">
    <slot />
  </div>

  <div v-else-if="slots">
    <slot name="header" />
    <slot name="results" />
    <slot name="pager" />
  </div>

  <div v-else>
    <DruxtEntity v-for="result of $attrs.results" :key="result.id" v-bind="props(result)" />
  </div>
</template>

<script>
export default {
  druxt: {
    ...
    query: {
      bundleFilter: true,
      fields: ['title'],
    }
  }
}
</script>

# DruxtJS

DruxtJS is a suite of modules to connect a Drupal (opens new window) JSON:API backend to a Nuxt.js (opens new window) frontend.

Find out more at http://druxtjs.org (opens new window)
